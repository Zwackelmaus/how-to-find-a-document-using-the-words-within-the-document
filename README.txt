# how-to-find-a-document-using-the-words-within-the-document
for the scatteredbrained like me, a good way to find a document on your computer.
there are two items, you can use either of them

if you use the item whose extension is "py", just use Spyder, Pycharm, or Jupyter notebook to open it

or you can open the docx file, copy the code, and paste it on the project in spyder, Pycharm or Jupyter notebook.
